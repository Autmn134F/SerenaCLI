# Serena Native Wrapper

A lightweight command-line interface for accessing Serena's semantic code understanding capabilities.

## Installation

```bash
pip install serena-native-wrapper
```

## Usage

### Initialize a Project

```bash
serena-native project init --project-root /path/to/repo
```

### Index the Project

```bash
serena-native project index --project-root /path/to/repo
```

### Query Symbols

```bash
serena-native query find-symbol --name "MyClass"
```
